In documentation I show all functions that you may using. To see tutorial, with example, go to this link.

slib.init(name, filename, enc) - Set name and default parameters of library for properly initialization.
Parameters:
name - string of name
filename (optional) - your default save file name
enc (optional) - default key for encription
Return:
nothing

slib.isFirst(filename) - Check, that save file is in folder. Use for first saving.
Parameters:
filename (optional) - your save file name
Return:
boolean - if it is first save

slib.save(table, filename, drop, enc) - Save table without encryption (example: maps, names…). It’s much faster.
Parameters:
table - table to save
filename (optional) - your save file name
drop (optional) - if you have custom userdata (like Image) in table
enc (optional) - key for encription
Return:
boolean - true while end saving

slib.save(table, filename, drop, enc) - Save table with encryption (example: stats, money…). It’s more slow than usual save.
Parameters:
table - table to save
filename (optional) - your save file name
drop (optional) - if you have custom userdata (like Image) in table
enc (optional) - key for encription
Return:
boolean - true while end saving

slib.load(filename, enc) - Load your table from file (can be encrypted, or not).
Parameters:
filename (optional) - your save file name
enc (optional) - key for encription
Return:
table - table that you save
boolean - true while end loading

slib.clear(filename) - Clear save file
Parameters:
filename (optional) - your save file name
Return:
boolean - true, if clear succsess
